eSIM Setup

eSIM 設定教學

建議出發前先安裝 eSIM，到泰國後先開啟數據漫遊及選用 TRUE eSIM 上網。

iPhone

iPhone 安裝方法

iOS
  1. 1. 打開「設定」>「流動服務」或「Mobile Service」。
  2. 2. 選擇「加入 eSIM」或「Add eSIM」。
  3. 3. 選擇「使用 QR Code」，掃描購買後收到的 eSIM QR Code。
  4. 4. 按指示加入 eSIM，名稱可改為「Thailand TRUE」。
  5. 5. 出發前先保持新 eSIM 關閉，到泰國後再開啟。
Android

Android 安裝方法

5G
  1. 1. 打開「設定」>「網絡和互聯網」或「連線」。
  2. 2. 進入「SIM」/「SIM 管理員」/「流動網絡」。
  3. 3. 選擇「加入 eSIM」或「下載 SIM」。
  4. 4. 掃描購買後收到的 QR Code，按手機提示完成安裝。
  5. 5. 不同品牌名稱略有不同，如 Samsung、Pixel、小米可按相近選項操作。
ON

到泰國後啟用

抵達泰國後，開啟 TRUE eSIM、選它作流動數據，並打開「數據漫遊」；落機即開網，叫車、睇地圖、搵酒店都即刻用得。

SIM

保留香港 SIM

原本香港 SIM 可以繼續放在手機內，建議只把上網數據切換到 TRUE eSIM，方便保留香港號碼收 SMS。

QR

QR Code 注意

QR Code 通常只可安裝一次。安裝後不要刪除 eSIM，除非已完成旅程或客服指示。

FAQ

常見問題

幫客人快速解決購買前最常見的疑問。

泰國 TRUE 5G eSIM 係咪無限流量？ +

係，全部方案都係 TRUE 5G eSIM。7日及10日方案為5G無限流量，高速用量後會降速；15日及30日方案為5G網絡全速不降速。

需要換實體 SIM 卡嗎？ +

唔需要。eSIM 透過 QR Code 加入手機，唔使拆原本香港 SIM。

使唔使實名登記？ +

完成網上購買後即可收到 QR Code，唔使去門市排隊換卡。實際使用如有手機或地區設定要求，請以結帳頁及客服通知為準。

有冇泰國本地電話號碼？ +

呢個方案主打數據上網，不包泰國本地電話號碼；不過 15日及30日方案支援免費接聽電話和收短信，適合保留香港號碼又想照樣上網的客人。

有電話或短信功能嗎？ +

全部方案均接聽免費。15日及30日方案支援免費接聽電話和收短信。

邊個方案最適合一般自由行？ +

10日 HK$52 方案最適合一般自由行，5G 50G高速流量夠用又抵玩；需要5G高速長時間使用就建議 15日或30日 5G全速方案。
